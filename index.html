EVAR: Edge Visual Autoregressive Models via Principled Pruning
Zefang Wang (Anonymous CVPR submission)
CVPR, 2026 (Under Review)
code / paper

An efficient structured-pruning framework tailored to next-scale Visual Autoregressive (VAR) models for edge deployment. EVAR combines Optimal Brain Surgeon-guided Hessian-aware sensitivity estimation with Progressive Scale-Aware Distillation (PSAD), achieving 1.8× speedup on iPad Pro (M4) with minimal quality degradation (FID: 3.55→3.91).

Key Achievements:
• Reduced latency from 494ms to 277ms on iPad Pro M4 (1.8× speedup)
• Compressed model from 310M to 230M parameters (26% reduction)
• Maintained competitive generative quality (FID increase < 10%)
• First principled pruning framework for next-scale VAR models
